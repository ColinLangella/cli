## SYNOPSIS
    ./nodectl cluster diff-tables CLUSTER_NAME NODE1 NODE2 TABLE_NAME <flags>
 
## DESCRIPTION
    Compare table on different cluster nodes
 
## POSITIONAL ARGUMENTS
    CLUSTER_NAME
    NODE1
    NODE2
    TABLE_NAME
 
## FLAGS
    -u, --use_checksums=USE_CHECKSUMS
        Default: False
    -b, --block_size=BLOCK_SIZE
        Default: 2

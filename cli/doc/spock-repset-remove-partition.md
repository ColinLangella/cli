## SYNOPSIS
    ./nodectl spock repset-remove-partition PARENT_TABLE DB <flags>
 
## DESCRIPTION
    Remove a partition from a replication set.
 
## POSITIONAL ARGUMENTS
    PARENT_TABLE
    DB
 
## FLAGS
    --partition=PARTITION
        Type: Optional[]
        Default: None
    --pg=PG
        Type: Optional[]
        Default: None

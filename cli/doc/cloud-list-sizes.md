## SYNOPSIS
    multi./pgedge cloud list-sizes PROVIDER <flags>
 
## DESCRIPTION
    List available node sizes.
 
## POSITIONAL ARGUMENTS
    PROVIDER
 
## FLAGS
    -r, --region=REGION
        Type: Optional[]
        Default: None
    -p, --project=PROJECT
        Type: Optional[]
        Default: None
    -j, --json=JSON
        Default: False

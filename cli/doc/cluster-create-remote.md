## SYNOPSIS
    ./nodectl cluster create-remote CLUSTER_NAME NUM_NODES <flags>
 
## DESCRIPTION
    Comiing Soon! Create a remote SSH cluster from a cluster json definition file.
 
## POSITIONAL ARGUMENTS
    CLUSTER_NAME
    NUM_NODES
 
## FLAGS
    --pg=PG
        Type: Optional[]
        Default: None
    -a, --app=APP
        Type: Optional[]
        Default: None
    --port1=PORT1
        Default: 6432
    -U, --User=USER
        Default: lcusr
    -P, --Passwd=PASSWD
        Default: lcpasswd
    -d, --db=DB
        Default: lcdb

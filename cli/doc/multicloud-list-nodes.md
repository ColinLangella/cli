## SYNOPSIS
    ./pgedge multicloud list-nodes PROVIDER <flags>
 
## DESCRIPTION
    List virtual machines.
 
## POSITIONAL ARGUMENTS
    PROVIDER
 
## FLAGS
    -r, --region=REGION
        Type: Optional[]
        Default: None
    -p, --project=PROJECT
        Type: Optional[]
        Default: None
    -j, --json=JSON
        Default: False

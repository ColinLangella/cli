## SYNOPSIS
    ./pgedge multicloud create-node PROVIDER REGION NAME <flags>
 
## DESCRIPTION
    Create a node.
 
## POSITIONAL ARGUMENTS
    PROVIDER
    REGION
    NAME
 
## FLAGS
    --size=SIZE
    
    
    -i, --image=IMAGE
    
    
    --ssh_key=SSH_KEY
    
    
    -p, --project=PROJECT
    
    
    -j, --json=JSON

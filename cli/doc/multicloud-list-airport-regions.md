## SYNOPSIS
    ./pgedge multicloud list-airport-regions <flags>
 
## DESCRIPTION
    List airport codes & corresponding provider regions.
 
## FLAGS
    -g, --geo=GEO
        Type: Optional[]
        Default: None
    -c, --country=COUNTRY
        Type: Optional[]
        Default: None
    -a, --airport=AIRPORT
        Type: Optional[]
        Default: None
    -p, --provider=PROVIDER
        Type: Optional[]
        Default: None
    -j, --json=JSON
        Default: False

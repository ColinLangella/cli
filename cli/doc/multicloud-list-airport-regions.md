## SYNOPSIS
    ./pgedge multicloud list-airport-regions <flags>
 
## DESCRIPTION
    List airport codes & corresponding provider regions.
 
## FLAGS
    -g, --geo=GEO
    
    
    -c, --country=COUNTRY
    
    
    -a, --airport=AIRPORT
    
    
    -p, --provider=PROVIDER
    
    
    -j, --json=JSON

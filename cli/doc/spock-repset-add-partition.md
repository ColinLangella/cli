## SYNOPSIS
    ./nodectl spock repset-add-partition PARENT_TABLE DB <flags>
 
## DESCRIPTION
    Add a partition to a replication set.
 
## POSITIONAL ARGUMENTS
    PARENT_TABLE
    DB
 
## FLAGS
    --partition=PARTITION
        Type: Optional[]
        Default: None
    -r, --row_filter=ROW_FILTER
        Type: Optional[]
        Default: None
    --pg=PG
        Type: Optional[]
        Default: None
